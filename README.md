# Github traffic viewer

> This is a full front-end application who simply rely on Github API. You can see it at [gitgraph.wellsguillaume.fr](https://gitgraph.wellsguillaume.fr).

## Installation

* `npm install`

## Server run

* `npm run server`

## Connect
Connect to :

* `http://127.0.0.1:9898/`
